## 简介
复深蓝DIME数智子午线自动填充周报脚本


> [!NOTE]
> 本脚本旨在解决复深蓝员工每月底手动填写周报的重复劳动问题
> 
> 它可以帮大家节省宝贵时间，把精力留给家人和朋友 :)
> 
> 本项目仅用于学习与交流，功能已能解决基本问题，但尚不完美。如你有更优方案，欢迎提 Issue 讨论


## 快速开始

- 修改参数
```
login_data = {
    "username": "你的工号",
    "cn_username": "你的芳名",
    "password": "你的密码",
}
```

- 安装依赖
```
pip install requests pillow
```

- 运行
```
python main.py
```

> [!WARNING]
> 当你运行脚本时，你需要查看 debug_captcha 文件夹下的验证码图片并输入到终端中。
>
> 建议步骤如下：
>
> 首先注释掉 submit_data 方法，仅运行填报功能，观察生成的日报数据是否正确
> 
> 确认数据无误后，将 submit_data 方法取消注释，并注释掉 post_weekly_data，运行提交周报部分

## 其余业务人员

> [!IMPORTANT]
> 目前默认的项目编号和任务类别是 `P24100054` 和 `程序技术`，如果需要修改，你可以自行修改 `post_weekly_data` 方法中的 `payload` 参数
> 
> 对于假期的填充目前尚不支持，你可以先利用脚本全部填充后手动在页面修改，再运行脚本提交。
